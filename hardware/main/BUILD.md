
Build Instructions

Install the components on the board in this order:

- Capacitors x31
- Resistors R5 and R6
- Resistors R11, 13, 14
- Resistors R12
- Resistor Array RP1
- Transistor 2N7000
- LEDs
- Sockets U3, U4, U5, U8, U9, U10
- U1, U2
- Capacitor 100uF
- Power sockets
- Sockets U6 and U7
